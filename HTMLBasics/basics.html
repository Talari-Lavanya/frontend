HTML Tags and Their Uses

Below is a brief description of the HTML tags used in this document.

  • <html> - Root element that contains all other HTML elements.
  • <head> - Contains metadata, title, and links to styles/scripts.
  • <body> - Contains the main content of the web page.
  • <h1> to <h6> - Headings of different levels.
  • <p> - Defines a paragraph.
  • <em> and <strong> - Italicize and bold text, respectively.
  • <a> - Anchor tag used for links.
  • <ul> and <li> - Unordered list and list items.
  • <blockquote> - Defines a blockquote section.
  • <hr> - Creates a horizontal rule.
  • <img> - Embeds an image.
  • <div> and <span> - Division and inline elements for grouping content.
  • <label> - Labels for form controls.
